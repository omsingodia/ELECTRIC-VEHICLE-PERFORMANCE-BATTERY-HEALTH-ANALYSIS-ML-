⚡ EV Battery Health Prediction
Predict electric vehicle battery health using real-world sensor & usage data — built as both a regression and classification problem.

🧠 What We Did

Regression → Predict exact Battery_Health_%
Classification → Healthy (1) or Unhealthy (0), using median as auto-balanced threshold
Compared 4 models each — Linear Regression, Random Forest, KNN, SVM
Added EDA: distribution plots, scatter plots, correlation heatmap
Fixed class imbalance via downsampling
